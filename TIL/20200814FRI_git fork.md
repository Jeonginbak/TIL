# git fork란 무엇일까?
git fork란 다른 원격 저장소에 있는 내용을 내 원격저장소로 복사하는 것을 말한다.
복사한 내용은 원본 저장소와 연결이 되어있으며 original 저장소의 내용이 수정될 경우
내가 복사한 저장소에 추가, 변경된 내용을 반영할 수 있고, 내가 복사한 저장소에서 추가, 변경한
내용도 원본 저장소에 반영할 수 있다.
